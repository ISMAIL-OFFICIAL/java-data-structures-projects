# Java Data Structures Projects

This repository contains two Java console applications developed to strengthen my understanding of Object-Oriented Programming (OOP), Data Structures, Algorithms, and File Handling. Each project focuses on solving a real-world problem while implementing core Java concepts and efficient data structures.

---

## Projects Included

### 1. Hospital Emergency Queue System

A hospital management simulation that prioritizes patients based on the severity of their condition using a `PriorityQueue` instead of treating them strictly by arrival time.

#### Features

- Add new patients
- Treat the highest-priority patient
- View the next patient in the queue
- View all waiting patients
- Search patients by name
- Display hospital statistics
- Save patient records to a file
- Load saved records automatically

#### Concepts Covered

- PriorityQueue
- Queue Operations
- Comparable Interface
- Searching
- File Handling
- Data Persistence
- Object-Oriented Programming (OOP)

---

### 2. Student Record Manager

A student management system that efficiently stores and manages student records using a `HashMap`, supporting complete CRUD operations and persistent storage.

#### Features

- Add student records
- Search students by roll number
- Update student information
- Delete student records
- View all students
- Display student statistics
- Save records to a file
- Load saved records automatically

#### Concepts Covered

- HashMap
- CRUD Operations
- Searching
- File Handling
- Data Persistence
- Object-Oriented Programming (OOP)
